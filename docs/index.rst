Overview
========

Qiskit-Qulacs allows user to execute Qiskit programs using Qulacs backend. Using `Qulacs <https://github.com/qulacs/qulacs>`_, a high-speed simulation library written in C++, the provider enhances Qiskit's capabilities for faster quantum circuit simulations.

This project has been made possible thanks to the support from `Unitary Fund microgrant program <https://unitary.fund/grants/>`_.
